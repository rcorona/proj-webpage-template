How it works: Because we assume that each pixel is of unit length, I have a double loop which loops over all x,y pairs from floor(xmin, ymin) to ceil(xmax, ymax) in increments of 1, where the mins/maxes are the minimum/maximum values out of each of {x0, x1, x2}, {y0, y1, y2} (ie. the bounding box of the triangle). For each of these pairs, I add 0.5, i.e. (sx, sy) = (x + 0.5, y + 0.5), so that we sample each pixel from the center.
